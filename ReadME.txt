Project: Music Recommender
Authors: Hansi Seitaj, Eni Vejseli, Andrew Kozempel
Date: 04/25/2022

Music recommender application using machine learning algorithms as
PCA and KNN.

This is the first version of the simulation. This version
does not use inheritance. The version uses a variety of libraries.

How to start:
	Please run these commands in your Anaconda promt:
		pip install spotipy
		pip install plotly
		pip install chart_studio
	Open a Spotify developer account (for free):
		Get unique client_id and client_secret IDs from Spotify Developers website.
